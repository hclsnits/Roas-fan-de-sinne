# @apps/search

Search service for the Munchies platform. This app is shaped for a lightweight Cloudflare Worker that can index products from Medusa and return storefront search results.

## Scripts

| Command | Description |
| ------- | ----------- |
| `pnpm dev` | Run worker locally |
| `pnpm build` | Type-check worker source |
| `pnpm deploy` | Deploy worker |
